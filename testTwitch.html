- Keyboard shortcuts?
F11 to fullscreen with chat.
Shift-Enter to add a stream without chat.
Ctrl to auto-close the menu after an action
(like adding a stream or using the controls).

- How do I select the link for my setup?
Right-click it and select "Copy link address / location / shortcut". On mobile, tap and hold to get the menu.

- Can I add multiple channels at once?
Yes! Just separate them by spaces, comas, dashes or forwardslashes. You can also use links from other multitwitch services and it will add the streams and/or chats for you! (suported services are the ones that separate the channel names by forwardslash). Please tell me if a specific link is not working.

- Can I use Twitch links as channel names?
Yes! Feel free to copy-paste them from chat!

- Stream hosting not working?
You have to add the stream that is being hosted. You can use the chat of your choosing though.

- Mobile/HTML5 mode?
When on mobile devices or when Flash Player is not found the site will fall back into HTML5 mode. This mode is experimental and some features dont yet fully work. If you are on a desktop/laptop you should install Flash Player from Adobe. Some of the features that dont yet fully work are viewer count and stream autoplay on mobile.

- Supported mobile devices/browsers?
Support varies. Chrome on Android seems to work just fine. Safari on iOS can only play one stream at a time. Keep in mind that this technology is evolving, so expect more and better support in the future. I will post any news I get on my twitter.

- Fullscreen looks small on my device.
I will try to work this around on a future patch.

- Clicking on streams navigates away?
Try using the new twitch player (in the "more options" menu). This is just something that the old flash player does when its embedded on other sites.

- TwitchTheater.tv link format?
Add the stream names after the website address, separated by forwardslashes. To specify the chats to load, just add a "/c/" at the end of the URL and then add the chat names after that (also separated by forwardslashes). If the chat parameter is omitted, then the site will add chats based on the stream names. Use an empty chat parameter to prevent chats from getting added. You can also use an empty chat parameter (without stream names) to prevent session saving from loading.

- How does session saving work?
It will store your streams and chats every time you add one and then restore them next time you open/reload the page. On TwitchTheater.tv links, it will load the streams/chats from the link instead (and will store them).

- Custom menu color?
Search Google for an "online color picker", generate the color of your choice, then copy the color code with a format like: #1a2b3c (with or without the hashtag). It is sometimes labeled as an RGB or an HTML color code.
